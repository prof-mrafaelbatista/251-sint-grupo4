Comandos Iniciais em Python

Veja alguns comandos básicos para começar a programar em Python:

  • print() — Exibe mensagens na tela.
    Exemplo: print('Olá, mundo!')
  • input() — Recebe dados do usuário.
    Exemplo: nome = input('Digite seu nome: ')
  • int(), float(), str() — Conversão de tipos.
    Exemplo: idade = int(input('Idade: '))
  • if, else, elif — Estruturas condicionais.
  • for, while — Estruturas de repetição.
  • def — Definição de funções.
  • import — Importação de módulos.

Esses comandos são a base para qualquer programa em Python. Pratique e experimente no seu próprio ambiente!
